# Hinterland Mobile Detailing — Landing Page

Professional mobile car detailing across the Sunshine Coast Hinterland (Maleny, Montville, Mapleton, QLD).

Static landing page — single `index.html`, single `styles.css`, single `app.js`, assets in `assets/`.

## Local preview

Open `index.html` in your browser, or run a quick local server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deployment

This site is deployed via **GitHub Pages** — every push to `main` updates the live site.

## Stack

- HTML + CSS + vanilla JS (no build step)
- Google Fonts: Fraunces (serif) + Manrope (sans)
- Brand colors: forest green `#016531` · beige `#F5F2E8`
